# 测试

程序在发布之前，要先经过所有功能的测试，在保证没有问题之后再发布

程序的两种使用方式：api和cli都要经过测试

`api`我自己在项目中通过一系列的ipynb就可以完成测试了

`cli`的测试比较麻烦，**因其功能较为繁杂，测试起来颇费时间，因此需要他人协助**

每次我写完代码需要测试时，都会把代码编译成可执行程序上传到 [蓝奏云](https://wwrz.lanzout.com/b00uyblcmh)（pywfn），其中记录的有程序的版本号和上传时间，测试人员需要下载下来解压缩运行，逐个测试每个功能是否能正常运行

> 因为打包后的可执行程序实际上运行的就是cli功能，因此如果测试完毕没有问题便可以提交正式版本了
